# 不规则火焰

**评分: ⭐⭐**

## 概念

锯齿状、风吹边缘，更接近真实赛车回火外观。不是圆形团块 - 而是不规则、流动的火焰。

## 用户反馈的要求

1. 不规则边缘（不是圆形/团块状）
2. 流动/飘动外观（被风吹）
3. 3D深度（明亮核心，较暗边缘）
4. 左右火焰大小不同
5. 帧间大小动态变化
6. 火焰必须从排气管中冒出
7. 水平拉长的喷射形态

## 关键技术

### 不对称尺寸
```python
# 左右独立的不同正弦周期
size_l = base_size * (0.8 + 0.4 * sin(t * 0.35))
size_r = base_size * (0.8 + 0.4 * sin(t * 0.45 + 1.3))
```

### 风吹边缘
不同频率的多层湍流创造不规则边界。

## 挑战

- 比平滑方法更难调节
- 极端设置下有伪影风险
- 需要平衡真实感和稳定性
